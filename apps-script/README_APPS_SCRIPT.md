# NFC Sheets Logger - Android 版（Apps Script）

PC 版と同じスプレッドシートに記録する、Android スマホ向け Web アプリです。

## セットアップ手順

### 1. Apps Script を開く

対象のスプレッドシートを開き：

```
メニュー → 拡張機能 → Apps Script
```

### 2. コードを貼り付け

Apps Script エディタが開いたら：

#### Code.gs（既存のファイルを上書き）

1. 左側の `コード.gs`（または `Code.gs`）をクリック
2. 中身をすべて削除
3. `Code.gs` の内容をコピー＆ペースト

#### index.html（新規ファイルを追加）

1. 左側の「＋」→「HTML」をクリック
2. ファイル名を `index` にする（.html は自動付与）
3. 中身をすべて削除
4. `index.html` の内容をコピー＆ペースト

### 3. デプロイ

1. 右上の **「デプロイ」→「新しいデプロイ」** をクリック
2. 歯車アイコン → **「ウェブアプリ」** を選択
3. 以下を設定：
   - **説明**: NFC Sheets Logger
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: 自分のみ（または組織内の全員）
4. **「デプロイ」** をクリック
5. **「アクセスを承認」** をクリック → Google アカウントで許可
6. 表示された **Web アプリの URL** をコピー

### 4. Android でホーム画面に追加

1. Android の **Chrome** で Web アプリの URL を開く
2. Chrome メニュー（︙）→ **「ホーム画面に追加」**
3. アイコン名を設定して追加

これでホーム画面からアプリのように起動できます。

## 使い方

```
1. ホーム画面のアイコンをタップ
2. 「NFC スキャン開始」ボタンをタップ（初回のみ）
3. 受話器の NFC タグにスマホをかざす
4. ボタンで入力を選択：
   - 変更あり
   - 変更なし
   - 疑義照会（→ 患者番号入力）
5. 5秒無操作で「変更あり」として自動記録
6. 記録完了 → 自動的に NFC 待機に戻る
```

## 動作要件

| 項目 | 要件 |
|------|------|
| スマホ | Android（NFC 搭載） |
| ブラウザ | Chrome 89 以上 |
| NFC タグ | NDEF 対応の汎用 NFC タグ（NTAG213 等） |

> **注意**: iOS の Safari は Web NFC に対応していません。Android Chrome のみです。

## PC 版との共存

- **同じスプレッドシート**、**同じシート**（NFC Logs）に記録されます
- PC 版と Android 版を同時に使用しても問題ありません
- どちらも末尾に行を追加するだけなので干渉しません

## コード更新時

コードを修正したら：

1. Apps Script エディタで保存
2. **「デプロイ」→「デプロイを管理」**
3. 鉛筆アイコン → バージョンを **「新バージョン」** に変更
4. **「デプロイ」** をクリック

> URL は変わりません。ブラウザの更新（またはホーム画面から再起動）で反映されます。
